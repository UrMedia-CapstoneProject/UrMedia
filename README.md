# UrMedia

UrMedia is an all-in-one media tracking website where users can search for, follow, rate, review, and create lists for movies, shows, anime, video games, and books. The goal of the project is to give users one place to manage different types of media instead of using separate apps for each category.

## Purpose

The purpose of UrMedia is to help users keep track of the media they are currently watching, playing, reading, or planning to start. Users can create an account, follow media, update their progress, leave ratings and reviews, and view profile features such as favorite media and personal stats.

## Technology Stack
- **Next.js** — framework used to build the web application
- **React** — used to create the user interface and reusable components
- **TypeScript** — used to improve code reliability
- **Node.js** — used for the project runtime and package management
- **Supabase** — used for the database, authentication, and security policies
- **Vercel** — used to deploy and host the website
- **TMDB API** — used to retrieve movie and TV show data
- **JIKAN API** - used to retrieve anime and manga data
- **RAWG API** — used to retrieve video game data
